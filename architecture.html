Data Flow
ActionFlow
LoginUser enters credentials → JS checks localStorage aq_users → sets aq_session → renders role-specific UI
Create GoalEmployee fills form → JS validates (weight, count, UoM) → writes to aq_goals → logs to aq_audit
Approve GoalManager clicks Approve → JS updates goal status + locked=true → saves to aq_goals → audit log entry
Log AchievementEmployee enters actual → JS calculates score via UoM formula → writes to aq_achievements
Escalation ScanAdmin clicks Scan → JS evaluates rules against aq_goals/aq_checkins → renders issues → logs to aq_esclog
AnalyticsAdmin views Analytics → JS aggregates aq_goals + aq_achievements + aq_checkins → renders Canvas charts
💡 Cost: This solution costs ₹0 to build and ₹0 to host. All technologies used (HTML, CSS, JavaScript, localStorage, GitHub Pages) are completely free and open. The single-file architecture (atomquest.html) means the entire portal is one file — easy to share, version, and demo.
